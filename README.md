# Kametori

Kametori helps you to write a benchmark test suite

## Installation

Add this line to your application's Gemfile:

    gem 'kametori'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kametori

## Usage

Kametori.limits = [ { tag: "tag", limit: 1} ]
